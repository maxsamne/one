Atelier Ledger, translated into the site’s current typography system.

This page documents the palette, type tokens, spacing habits, and reusable components used across the site. It keeps the warmer editorial identity while using Lora, Inter, and JetBrains Mono.

Type tokens

Lora for headings that need warmth and gravity.

Layout

Rules that should stay stable

  • Homepage stays sparse: name, one line of positioning, and 2–3 links.
  • Writing pages remain prose-first and keep the 680px article body width.
  • Body copy on essays stays at roughly 1.8 line-height.
  • Use borders and subtle blur before adding ornament.
  • Keep accent color usage deliberate and low-volume.
